# Tutor Website

A responsive educational website for showcasing tutors, their expertise, and course content. Built as part of my frontend learning journey, the Tutor website focuses on clean UI, accessibility, and performance.

---

## Live Demo

([https://lotacodic.github.io/tutor-course-builder/](https://lotacodic.github.io/tutor-course-builder/))

---

## Features

- **Tutor Pages** – Author information with background, expertise, experience, and skills
- **Course Content Sections** – Structured, readable layouts for learning materials
- **Responsive Design** – Mobile-first, works across devices
- **Reusable Components** – Clean, modular HTML/CSS structure
- **Modern Layouts** – Grid/Flexbox for alignment and spacing

---

## Built With

- **HTML5** – Semantic structure
- **CSS3** – Flexbox, Grid, BEM methodology
- **JavaScript (Vanilla)** – Interactivity (modals, UI behavior)
